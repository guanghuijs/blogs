  • 可选参数

main(List<String> args) {

void sayHello1(String name) {

void sayHello2(String name, [int age = 10, double height = 2]) {

}

// 命名可选参数
void sayHello3(String name, {int age = 10, double height = 3.14}) {

}

  • 函数是一等公民
dart
main(List<String> args) {
  // 1.直接找到另外一个定义的函数传进去
  // test(bar);

  // 2.匿名函数 (参数列表) {函数体};
  test(() {
    print("匿名函数被调用");
    return 10;
  });

  // 3.箭头函数: 条件, 函数体只有一行代码
  test(() => print("箭头函数被执行")); 
}

// 函数可以作为另外一个函数的参数
void test(Function foo) {
  var result = foo();
}


void bar() {

运算符 ​

  • ??=

当原来的变量有值时, 那么??=不执行,原来的变量为null, 那么将值赋值给这个变量

dart
var name = null;
name ??= "lilei"; // lilei
1
2
  • ??

??前面的数据有值, 那么就使用??前面的数据,??前面的数据为null, 那么就使用后面的值

dart
var name = null;
var temp = name ?? "lilei"; // lilei

  • 循环

  }
}
